Sound Files for Master Dashboard
================================

Place the following audio files in this directory:

1. notification.mp3 - Default notification sound for "waiting for input" status
2. success.mp3 - Optional sound for successful operations
3. error.mp3 - Optional sound for error states
4. warning.mp3 - Optional sound for warnings

Requirements:
- Format: MP3
- Duration: 0.5 - 2 seconds
- File size: Under 100KB each

Recommended sources for free sounds:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.soundjay.com/

License Note:
Ensure any sounds used have appropriate licensing for your use case.
